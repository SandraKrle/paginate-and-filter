## Description

A web app that shows all the starwars characters, which can be filtered by movies they star in. 

Build in Svelte

Data fetched from swapi.dev

## Get started

Install and run the project

```bash
npm install
```

```bash
npm run dev
```

Navigate to [localhost:5000](http://localhost:5000)
